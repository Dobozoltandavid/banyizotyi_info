Windows Page

Ez az oldal a Windows témakört mutatja be. Itt találhatók információk a Windows operációs rendszerről, parancsokról, disztribúciókról, és egyéb hasznos tudnivalókról.

Feladat: IP-cím beállítása, gépnév módosítása

1. Vezérlőpult → Hálózati és megosztási központ → Adapterbeállítások módosítása 2. Ethernet → Tulajdonságok → Internet Protocol 4 (TCP/IPv4) 3. Állítsd be: - IP-cím: 192.168.50.1 - Alhálózati maszk: 255.255.255.0 - Alapértelmezett átjáró: 192.168.50.254 - DNS: 127.0.0.1 és 8.8.8.8 4. Gépház → Rendszer → Név → Számítógép átnevezése: SandBox → újraindítás

Feladat: Szerepkörök telepítése

Kiszolgálókezelő → Szerepkörök és szolgáltatások hozzáadása: - Active Directory tartományi szolgáltatások - DNS-kiszolgáló - DHCP-kiszolgáló → Tovább → Telepítés

Feladat: Tartományvezérlővé előléptetés

A telepítés után: - Kattints: „A kiszolgáló előléptetése tartományvezérlővé” - Új erdő létrehozása: box.local - DSRM-jelszó: P@ssw0rd - Egyéb beállítás: alapértelmezett

Feladat: MMC konzol készítése

1. Parancssor → mmc → OK 2. Fájl → Beépülő modul hozzáadása: - Active Directory – felhasználók és számítógépek - DHCP - DNS 3. Fájl → Mentés másként → Asztalra: DNS-DHCP.msc

Feladat: Szervezeti egységek létrehozása

1. Nyisd meg az Active Directory – felhasználók és számítógépek konzolt. 2. Kattints jobb gombbal a vizsga.local tartományra → Új → Szervezeti egység. 3. Add meg a neveket: vizsgaszervezők és informatika.

Feladat: Felhasználó létrehozása

1. A nyiff nevű szervezeti egységen kattints jobb gombbal → Új → Felhasználó. 2. Vezetéknév: Vicc, Keresztnév: Elek, Bejelentkezési név: elek. 3. Jelszó: P@ssw0rd 4. Jelöld be: „A felhasználónak nem kell megváltoztatnia a jelszót” és „A jelszó soha ne járjon le”.

Feladat: Csoportok létrehozása

1. A nyiff nevű szervezeti egységen kattints jobb gombbal → Új → Csoport. 2. Hozd létre: - nyuff (Globális, Biztonsági) - mail (Tartományon belüli, Terjesztési)

Feladat: Csoporttagságok beállítása

1. Nyisd meg a Vicc Elek felhasználót → "A következő tagja" fül → Hozzáadás → nyuff. 2. Nyisd meg a nyuff csoportot → "A következő tagja" fül → Hozzáadás → mail.

Feladat: Chrome telepítés GPO-val

1. Töltsd le a Google Chrome MSI fájlt. 2. Másold egy megosztott mappába, pl. \\Szerver\megosztas 3. GPO szerkesztő → Számítógép konfigurációja → Házirendek → Szoftverbeállítások → Szoftvertelepítés 4. Új csomag → válaszd ki a .msi-t a hálózati útvonalon 5. Telepítés típusa: Kötelező

Feladat: Házirend frissítése

1. Nyisd meg a Parancssort vagy PowerShell-t. 2. Írd be: gpupdate /force 3. Nyomj Enter-t és várd meg a folyamat végét.

Feladat: DHCP hatókör létrehozása

1. Server Manager → DHCP → DHCP konzol 2. Jobb klikk IPv4-en → Új hatókör: - Név: sand - IP-tartomány: 192.168.50.20 – 192.168.50.120 - Kizárási tartomány: 192.168.50.50 – 192.168.50.70 - Alhálózati maszk: 255.255.255.0 - Címbérlet élettartama: 12 óra

Feladat: DHCP opciók beállítása

A DHCP kiszolgálóban állítsd be: - 003 Útválasztó: 192.168.50.254 - 006 DNS-kiszolgálók:     - Elsődleges: 192.168.50.1     - Másodlagos: 8.8.8.8

Feladat: DNS rekord létrehozása

1. Kiszolgálókezelő → DNS → box.local zóna 2. Jobb klikk → Új állomás (A rekord): - Név: fileserver - IP-cím: 192.168.50.2 3. Mentés → Ellenőrzés nslookup paranccsal
